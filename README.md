# Одностраничный сайт Елены Перелыгиной

Статический адаптивный лендинг на HTML, CSS и JavaScript. Не требует сборки.

## Локальный запуск

Откройте `index.html` в браузере или запустите любой локальный HTTP-сервер.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Добавьте его как `origin` и отправьте ветку `main`.
3. В настройках репозитория откройте **Pages → Deploy from a branch → main / root**.

Перед публикацией подключите реальный обработчик формы в `script.js` (CRM, Telegram-бот, email-сервис или backend).
